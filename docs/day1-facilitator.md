# Day 1 강사 진행 대본

> 강의 준비할 때 펼쳐놓고 보는 문서다. 참가자용이 아니라 **진행자(현종님·예찬님)용**이다.
> Day 1의 실제 내용은 `/day1` 스킬이 참가자에게 진행한다. 이 문서는 강사가 "각 블록에서 뭘 하고, 참가자가 어디서 막히는지"를 미리 아는 용도다.

## 90분 시간 배분

| 블록 | 주제 | 시간 | 성격 |
|------|------|------|------|
| 0 | Setup — 첫 실행 확인 | 5분 | 준비 |
| 1 | Experience — 콜 프렙 라이브 체험 | 12분 | 체험 |
| 2 | Why — 왜 지금 AI (인트로 3막) | 10분 | 스토리 |
| 3-1~3-4 | 핵심 기능 4개 (Memory·Skill·MCP·Subagent) | 26분 | 개념 |
| **4** | **CLAUDE.md 세팅 (오늘 산출물)** | **25분** | **실습·핵심** |
| 5 | Wrap — 자가진단 + 숙제 + Day 2 예고 | 7분 | 마무리 |

합 85분 + 버퍼 5분. **Block 4(CLAUDE.md)에 가장 많이 배분하고, 앞 체험·개념 블록은 가볍게 흘린다.**

## 전 블록 공통 — STOP PROTOCOL

각 블록은 2턴이다. 강사는 이 리듬을 알고 있어야 참가자가 "다음"을 언제 치는지 안다.

- **Phase A**: Claude가 설명(EXPLAIN) → "지금 직접 실행하세요"(EXECUTE) 안내 → **멈춤**
- 참가자가 터미널에서 실행 → "완료" 또는 "다음" 입력
- **Phase B**: 퀴즈/체크 → 다음 블록

Phase A에서는 퀴즈·질문을 하지 않는다. 각 블록 시작에 `📖 공식 문서: URL`이 그대로 출력된다.

## Block 0 — Setup (5분)

- **🎤 강사**: "지금 이 터미널 화면이 Claude Code다. 오늘 여기 하나로 다 한다"를 짚어준다. 설치는 어젯밤 숙제(README Step 0)로 끝난 전제.
- **⌨️ 참가자**: ① `/config` → Output style → **Explanatory** ② `안녕, 나는 [이름]이고 세일즈야. 인사해줘` ③ `지금 내가 어느 폴더에서 너를 실행하고 있어?` → `ai-camp-sales` 나오면 정상
- **✋ STOP**: 폴더 확인 명령 안내 후 멈춤. 퀴즈 없음.
- **⚠️ 막히는 곳**: 설치·로그인 실패 → claude.ai에 "Claude Code 설치하는 방법 알려줘" 입력해 환경별 안내. **90%는 재로그인 또는 재시작으로 해결.** 폴더가 `ai-camp-sales`가 아니면 이동 후 재실행. 에디터(Cursor/VSCode)는 오늘 불필요.
- **🖥 슬라이드**: slide-01 커버로 오프닝 → slide-02 지도로 오늘 4블록 먼저 보여주기

## Block 1 — Experience (12분)

- **🎤 강사**: 핵심 메시지 — **"맥락 없이도 이 정도가 나온다. 4일 뒤엔 여러분 도구가 연결된 버전이 된다."** Before/After(지금=스레드·노션·메일 수동 확인 → 4일 뒤=계정명 하나로 자동 브리핑). "지금은 도구 0개, 그래도 어디까지 나오나 본다."
- **⌨️ 참가자**: ① 다음 주 실제 미팅 1개 선택(실명 부담되면 "글로벌 커머스 앱 A사"로 익명화) ② 틀에 붙여넣기(계정/미팅 성격/참석자/아젠다/아는 배경 — 아는 만큼만) ③ **나온 브리핑을 고치지 말고 관찰**
- **✋ STOP**: 붙여넣기+관찰 안내 후 멈춤
- **✅ 체크**: 정답 없는 소감. "도구 0개여도 구조는 나온다 / 내 딜의 진짜 맥락은 비었다 = 4일의 과제"를 먼저 정리해준 뒤 소감 질문
- **⚠️ 막히는 곳**: "70%만 쓸 만해도 정상"을 미리 프레이밍(실망 방지). 브리핑 고치려는 사람 → "고치지 마라, 관찰이 진짜 실습"
- **🖥 슬라이드**: slide-02 지도 카드 ①

## Block 2 — Why (10분)

- **🎤 강사**: 숫자·FOMO 없이 **이야기 톤**으로 3막. **1막(왜 써야)**: 메일 3분의 환상(후속작업 합치면 7~8분) · KTX vs 택시(빠른 게 아니라 셋업 후 한계비용이 낮다). **2막(왜 배워야)**: "역시 AI는 여기까지" vs 디버깅 마인드셋 · 맥락의 두께 = 복리. **3막(왜 내가 직접)**: "내가 해야?" → "AI가 왜 이걸 안 하지?"로 전제 뒤집기.
- **⌨️ 참가자**: 듣고 곱씹기. 터미널 행동 없음.
- **✋ STOP**: 3막 후 "세 가지 왜를 곱씹어보세요" 문구로 멈춤
- **✅ 체크**: 정답 없는 reflection. "세 가지 '왜' 중 지금 가장 와닿는 건?"
- **⚠️ 막히는 곳**: **"안 쓰면 뒤쳐진다" 식 압박 금지** — 파일에 명시된 톤 가드레일. 압박형으로 흐르지 않게.
- **🖥 슬라이드**: slide-02 지도 카드 ②

## Block 3-1~3-4 — 핵심 기능 4개 (26분)

네 블록 각각 Phase A(설명+실행) → Phase B(퀴즈). 개념만 잡고 가볍게 흘린다. **Day 2~3에 직접 만들 것이라 오늘은 "뭔지만".**

| 블록 | 강사 한 줄 | 참가자 실행 | 퀴즈 정답 |
|------|-----------|-------------|-----------|
| **3-1 Memory** (7분) | CLAUDE.md(내가 쓰는 지시문, 매 대화 자동 로딩=팀 위키) vs Auto Memory(Claude가 쓰는 메모=업무 수첩, 기본 ON) | `/memory` 확인 → "나는 국내 커머스·핀테크 담당 세일즈야. 기억해둬" → `/memory` 재확인 | 작성 주체가 다르다 (내가 vs Claude가) |
| **3-2 Skill** (6분) | 점진적 로딩 — CLAUDE.md는 항상, Skill은 필요할 때만. "지금 이 `/day1`도 Skill" | `/day1-test-skill` 실행 | 필요할 때만 점진적 로드 |
| **3-3 MCP** (6분) | 외부 도구 플러그(Slack·Notion·Circleback 꽂는 규격). **"Block 1 콜프렙 맥락이 비었던 이유 = 아직 도구 안 꽂힘. 그게 Day 2."** | "MCP가 뭔지 쉽게 + 세일즈용 서버 예시 3개" 질문 | 외부 도구 연결 표준 |
| **3-4 Subagent** (7분) | 새 프로세스를 빈 상태로 띄워 병렬 처리(부하 직원). "Day 2 리서치 스킬이 이 구조" | "이 폴더 구조 탐색해줘. Explore 에이전트 써" | 독립 공간 병렬 처리 |

- **🔚 Block 3-4 끝 필수 브릿지 멘트**: "핵심 4개 다 봤어요. Hook·Plugin·Agent Teams는 더 있지만 오늘은 이 4개면 충분, 나머지는 뒤쪽 Day에서. 이제 오늘의 진짜 산출물 — 내 CLAUDE.md를 만들러 갑니다."
- **⚠️ 막히는 곳**: Memory 두 종류 헷갈림 → "작성 주체가 다르다". MCP를 내장 기능으로 오해 → 외부 연결 강조, 오늘 연결 안 함(밤 숙제).
- **🖥 슬라이드**: slide-02 지도 카드 ③ (4개 기능이 이 한 카드)

## Block 4 — CLAUDE.md 세팅 (25분 · 오늘의 목적지)

- **🎤 강사**: Phase A에서 개념 한 문단 재확인 + **인터뷰할 4가지 예고**(①담당 계정/세그먼트 ②톤/시그니처 ③금지어/주의사항 ④자주 하는 작업 3개). 이 블록만 예외 — Phase A에서 도구 안 쓰고, Phase B(인터뷰)에서 Claude가 적극적으로 묻고 저장한다.
- **⌨️ 참가자**: "시작" 입력 → 한 번에 한 주제씩 대화형 답변(세그먼트는 선택지, 계정·톤·서명은 자유서술) → 저장된 CLAUDE.md 확인
- **✋ STOP**: 4가지 예고 후 "준비되면 '시작'"
- **✅ 체크**: 퀴즈 없음. 인터뷰 4주제 → **Write로 `ai-camp-sales/CLAUDE.md` 실제 저장** → 산출물 확인
- **⚠️ 막히는 곳**: 답이 얇으면 **한 번만 더 캐묻기(0.5판 목표, 취조 금지)**. 실명 부담 → 익명 허용 미리 고지. **확정 안 된 할인/약속을 금지어에 넣도록 유도**(회사 정책 직결). 기존 CLAUDE.md 있으면 덮어쓰지 말고 "이어붙일까요, 새로?" 물어보기. "완벽 불필요, 0.5판이면 합격."
- **🖥 슬라이드**: slide-02 지도 카드 ④ + "오늘 손에 남는 것 = CLAUDE.md 한 장"

## Block 5 — Wrap (7분)

- **🎤 강사**: 오늘 4기능 관계도로 되짚기(CLAUDE.md 기본값 → Skill 반복업무 → MCP 외부도구 → Subagent 병렬수집). "내 CLAUDE.md 한 장이 이제 모든 대화의 출발점."
- **⌨️ 참가자**: "다음" → 자가진단 3문항 → 숙제 안내
- **✋ STOP**: 오늘 정리 후 "확인했으면 다음"
- **✅ 체크**: 자가진단 3문항 연속 — ① 담당계정 섹션이 구체적인가 ② 금지어에 '왜 안 되는지' 이유가 붙었나 ③ 4개 중 'Day 2 주인공' = **MCP**
- **⚠️ 막히는 곳**: 자가진단 1에서 "엔터프라이즈"만 있으면 → "연 계약·분기 QBR·트래킹 오차"처럼 구체 이슈를 붙이면 AI가 훨씬 잘 씀. **숙제 못 해와도 괜찮다**(Day 2 첫 15분이 연결 점검) — 시도가 핵심.
- **🖥 슬라이드**: slide-03 마무리 (오늘 만든 것 / 숙제 / Day 2 예고)
