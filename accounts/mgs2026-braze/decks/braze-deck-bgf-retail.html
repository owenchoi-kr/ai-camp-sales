분리돼 있던 데이터·채널·오케스트레이션을 하나의 회원 관점 위에서 돌립니다.

01

Canvas 저니 오케스트레이션

설치·첫 방문·재방문·로열티 전환을 하나의 시나리오로 잇는 라이프사이클 캠페인을 드래그앤드롭으로 설계.

02

실시간 트리거 + AI

매장 방문·적립·쿠폰 사용이 일어나는 순간 반응하고, BrazeAI가 채널·시간·소재를 회원별로 최적화.

03

개방형 데이터 구조

SDK·API·CDI·Currents로 POS·멤버십 시스템부터 웨어하우스·분석까지 데이터가 끊기지 않음.

핵심 기능

크로스채널 캠페인

앱푸시·인앱·이메일·SMS·Webhook(알림톡 등)을 하나의 여정으로 통합 운용. 단일 채널보다 여러 채널을 조합할수록 회원 활성화·재방문이 좋아지도록 설계합니다.

Canvas 시나리오

실시간 트리거

매장 방문·쿠폰 다운로드·적립 등 행동이 일어나는 순간 API Triggered Delivery로 즉시 발송.

AI Intelligent 개인화

Intelligent Channel·Timing·Selection으로 채널·시간·소재를 회원별 자동 최적화.

속성·Connected Content

멤버십 등급·최근 방문 매장·선호 카테고리 속성 + POS·주문 시스템 API 연동으로 추천 혜택을 메시지에 실시간 삽입.

Q&A

자주 묻는 질문

기존 카카오 알림톡·문자와 같이 쓸 수 있나요?
네. Canvas에서 앱푸시·인앱·이메일·SMS·Webhook(알림톡 등)을 하나의 시나리오로 통합 운용하고, 채널 간 시간차 순차 발송으로 효과를 높일 수 있습니다. 적립·쿠폰 안내처럼 알림톡이 강한 영역과 마케팅 푸시를 한 여정에서 함께 설계할 수 있습니다.
POS·멤버십 시스템 데이터와 연동되나요?
SDK·API로 방문·구매·적립 이벤트를 수집하고, Connected Content로 POS·멤버십 시스템 API의 매장·혜택·추천 데이터를 메시지에 실시간 반영합니다. Cloud Data Ingestion(CDI)으로 Snowflake·BigQuery·Redshift·Databricks를 최대 15분 주기로 동기화할 수도 있습니다.
메시지 개인화는 어디까지 되나요?
이름·선호 매장·멤버십 등급·동적 이미지·딥링크 등 프로필·행동 속성으로 개인화하고, Connected Content로 실시간 쿠폰·추천 혜택까지 메시지에 반영합니다.
도입까지 얼마나 걸리나요?
SDK 설치·이벤트 분류체계 설계부터 기본 캠페인 런칭까지 단계별로 지원합니다. 초기 세팅 이후에는 마케팅팀이 직접 운영할 수 있습니다.
개발 리소스가 많이 드나요?
초기 세팅 이후에는 CRM·마케팅팀이 SQL·추가 개발 없이 직접 캠페인을 런칭·자동 최적화할 수 있어 반복 작업과 개발 의존도를 크게 줄입니다.
보안·개인정보는 어떻게 관리되나요?
엔터프라이즈 보안·데이터 거버넌스 기능과 채널별 수신동의·옵트아웃 관리, 데이터 접근 권한 제어를 제공합니다. 리테일 특성상 필요한 규제·보안 요건은 도입 단계에서 함께 점검합니다.
성과는 어떻게 측정하나요?
캠페인별 오픈·클릭·전환·리텐션(재방문)을 대시보드로 추적하고, A/B·홀드아웃(대조군)으로 순증 효과를 검증합니다. Amplitude와 함께 쓰면 행동 분석까지 이어 붙여 "왜 좋아졌는지"를 더 깊게 볼 수 있습니다.
