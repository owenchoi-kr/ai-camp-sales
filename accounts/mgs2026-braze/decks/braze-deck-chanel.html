샤넬에 이렇게

고객과의 관계를 한 걸음씩 이어가는 여정

관심 등록부터 재구매까지, 고객마다 다른 취향과 타이밍에 맞춰 Braze Canvas로 흐름을 설계할 수 있습니다. 접점마다 절제되고 정제된 톤을 유지하는 것이 핵심입니다.

01 관심 등록취향 파악가입·매장 방문 시 남긴 관심 카테고리와 선호를 바탕으로 고객 프로필을 축적합니다.
02 신제품 소식컬렉션 프리뷰 안내관심 카테고리에 맞춰 신제품·컬렉션 소식을 우선 안내합니다.
03 초대부티크·이벤트 개인 초대VIP·고관여 고객에게 부티크 방문이나 프라이빗 이벤트를 개인화된 메시지로 초대합니다.
04 VIP 로열티재구매 & 멤버십구매 이력과 선호를 바탕으로 다음 컬렉션·리오더 시점을 자연스럽게 안내합니다.

위 흐름은 Braze Canvas로 구현 가능한 시나리오 예시이며, 실제 설계는 샤넬 데이터·목표에 맞춰 함께 정의합니다.

왜 Braze인가

따로 흩어져 있던 데이터·채널·오케스트레이션을 하나의 고객 관점 위에서 절제되게 운영합니다.

01

Canvas 저니 오케스트레이션

관심 등록부터 재구매까지 이어지는 고객 여정을 하나의 시나리오로 설계.

02

실시간 트리거 + AI

재입고·컬렉션 공개 등의 순간에 반응하고, BrazeAI가 채널·시간·소재를 고객별로 조율.

03

개방형 데이터 구조

SDK·API·CDI로 매장·이커머스·CRM 데이터가 끊기지 않고 하나로 이어짐.

핵심 기능

고객 커뮤니케이션에 필요한 모든 것을 한 플랫폼에서

크로스채널 캠페인

앱푸시·인앱·이메일·SMS·Webhook(알림톡 등)을 하나의 여정으로 통합 운용. 채널을 따로 운영하지 않고 고객 한 분의 맥락 위에서 순서·시간차까지 함께 설계합니다.

Canvas 시나리오

신제품 안내 → 관심 표시 → 부티크 초대 → 이후 팔로업처럼 시간차를 둔 크로스채널 흐름을 설계.

실시간 트리거

위시리스트 상품 재입고, 한정 컬렉션 공개 등 행동이 일어나는 순간 API Triggered Delivery로 전달.

AI Intelligent 개인화

Intelligent Channel·Timing·Selection으로 채널·시간·소재를 고객별 자동 최적화.

속성·Connected Content

프로필·구매이력 속성 + CRM/이커머스 API 연동으로 추천 상품·컬렉션을 메시지에 실시간 삽입.

NEW Cloud Data Ingestion

Snowflake·BigQuery·Redshift·Databricks를 최대 15분 주기 동기화, Connected Sources 제로카피.

NEW BrazeAI Operator

목표만 주면 세그먼트·타이밍·콘텐츠·A/B를 실시간으로 스스로 조정하는 Agentic AI.

라이프사이클 자동화

가입→첫 구매→단골 고객→VIP까지 전 구간을 자동화된 여정으로 운영.

반복 & Frequency Capping

반복 주기·고객별 제어·게재빈도 제한으로 고객 피로도까지 세심하게 조율.

Q&A

자주 묻는 질문

기존 카카오 알림톡·문자와 같이 쓸 수 있나요?
네. Canvas에서 앱푸시·인앱·이메일·SMS·Webhook(알림톡 등)을 하나의 시나리오로 통합 운용하고, 채널 간 시간차 순차 발송으로 효과를 높일 수 있습니다. 부티크 예약 안내처럼 알림톡이 강한 영역과 마케팅 메시지를 한 여정에서 함께 설계할 수 있습니다.
매장·이커머스·CRM 데이터와 연동되나요?
SDK·API로 구매·방문·멤버십 이벤트를 수집하고, Connected Content로 CRM/이커머스 API의 추천 상품·컬렉션 데이터를 메시지에 실시간 반영합니다. Cloud Data Ingestion(CDI)으로 Snowflake·BigQuery·Redshift·Databricks를 최대 15분 주기로 동기화할 수도 있습니다.
메시지 개인화는 어디까지 되나요?
이름·관심 카테고리·멤버십 등급·동적 이미지·딥링크 등 프로필·행동 속성으로 개인화하고, Connected Content로 실시간 추천 상품·컬렉션까지 메시지에 반영합니다.
도입까지 얼마나 걸리나요?
SDK 설치·이벤트 분류체계 설계부터 기본 캠페인 런칭까지 단계별로 지원합니다. 초기 세팅 이후에는 마케팅팀이 직접 운영할 수 있습니다.
개발 리소스가 많이 드나요?
초기 세팅 이후에는 CRM·마케팅팀이 SQL·추가 개발 없이 직접 캠페인을 런칭·최적화할 수 있어 반복 작업과 개발 의존도를 크게 줄입니다.
보안·개인정보는 어떻게 관리되나요?
엔터프라이즈 보안·데이터 거버넌스 기능과 채널별 수신동의·옵트아웃 관리, 데이터 접근 권한 제어를 제공합니다. 럭셔리 브랜드 특성상 필요한 고객정보 보호 요건은 도입 단계에서 함께 점검합니다.
성과는 어떻게 측정하나요?
캠페인별 오픈·클릭·전환·리텐션을 대시보드로 추적하고, A/B·홀드아웃(대조군)으로 순증 효과를 검증합니다. Amplitude와 함께 쓰면 행동 분석까지 이어 붙여 "왜 좋아졌는지"를 더 깊게 볼 수 있습니다.
