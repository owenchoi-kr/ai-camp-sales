# Block 2: Pick — 템플릿 4종 중 하나 고르기

> 0에서 만들지 않는다. 운영진이 만들어 둔 스킬 템플릿 4종 중 **내 주간 루틴에서 제일 시간 먹는 것** 하나를 고른다.
> 선택 블록: Phase A는 소개 + 선택 기준 안내 후 Stop, Phase B에서 AskUserQuestion으로 택1한다.

### 왜 템플릿에서 시작하나 — 0.5 → 1

스킬을 백지에서 만들면 시간도 오래 걸리고 빠지는 것도 많다. 그래서 오늘은 **운영진이 미리 만든 0.5짜리 템플릿**을 받아, 내 딜 맥락 몇 곳만 고쳐 **1로 완성**한다. 다음 블록에서 딱 3곳만 손대면 내 것이 된다.

### 오늘 고를 수 있는 템플릿 4종

| 템플릿 | 하는 일 | 이런 사람에게 |
|--------|---------|--------------|
| **call-prep** | 계정명 + 미팅 성격을 넣으면 → 콜 프렙 브리핑을 만든다 | 미팅 준비에 매번 시간 쓰는 사람 |
| **battlecard** | 계정명 + 경쟁 MMP + 갱신 시기를 넣으면 → 이관 설득 카드를 만든다 | 경쟁사 이관·갱신 딜이 많은 사람 |
| **meeting-followup** | 미팅 노트를 넣으면 → 액션아이템 + CRM 업데이트 제안 + 국/영문 팔로업 메일 초안을 만든다 | 미팅 후 팔로업이 자꾸 밀리는 사람 |
| **draft-outreach** | 프로스펙트 정보를 넣으면 → 아웃리치 초안 2버전을 만든다 | 신규 아웃리치를 자주 쓰는 사람 |

### 고르는 기준 — "제일 시간 먹는 것" 하나만

멋져 보이는 걸 고르지 않는다. **내 주간 루틴에서 지금 손으로 하느라 제일 시간이 많이 드는 것** 하나를 고른다. 오늘은 1개만 만든다. 나머지는 2주차 프로젝트에서 만들면 된다.

```
질문 셋으로 좁히기:
1. 이번 주에 이 작업을 몇 번 했나? (자주 할수록 스킬 효과 큼)
2. 매번 백지에서 시작하나? (그렇다면 템플릿 효과 큼)
3. 지금 내가 손으로 하고 있나? (수동일수록 자동화 가치 큼)
```

## EXECUTE

위 4종을 다시 훑어보고, **어느 게 내 주간 루틴에서 제일 시간을 먹는지** 잠시 떠올려본다. 답이 정해지면 다음 턴에서 함께 확정한다.
