# Block 5: Wrap — 자가진단 + Day 4 예고

> 오늘을 정리하고, 자가진단 3문항으로 산출물을 점검하고, 내일을 예고한다.

## EXPLAIN (Phase A)

오늘 한 걸 한 화면으로 되짚는다. (Block 5는 공식 문서 URL을 출력하지 않는다.)

**오늘 바뀐 3가지:**

```
① "바로 시켰다"  →  "묻고 시킨다"        Clarify · GIGO 체험 (Block 1)
② "매번 백지에서"  →  "템플릿을 내 것으로"   0.5→1 커스터마이징 (Block 3)
③ "일반적인 결과"  →  "내 톤·맥락이 묻어남"   실제 딜 실행 (Block 4)
```

**오늘의 산출물:** 슬롯 3곳을 내 맥락으로 채운 **동작하는 스킬 1개** + 내일 쓸 **아쉬운 점 메모 1줄**. Block 1에서 모호하게 시켜 나왔던 밋밋한 결과와, 오늘 만든 스킬이 내는 결과는 다르다. 그 차이를 만든 건 AI가 아니라 **"잘 시킨 나"**다.

Phase A 종료 문구:

```
---
👆 오늘 정리를 확인했으면 "다음"이라고 입력해주세요.
자가진단 퀴즈 3개와 Day 4 예고를 안내합니다.
```

여기서 Stop한다.

## QUIZ (Phase B) — 자가진단 3문항 연속

사용자가 돌아오면 3문항을 **연속으로** 출제한다. 정답을 맞히는 시험이 아니라, 오늘 만든 게 살아있는지 점검하는 자가진단이다. 각 문항에 짧은 피드백을 준다.

### 자가진단 1

```json
AskUserQuestion({
  "questions": [{
    "question": "내가 만든 스킬, `/<이름>`으로 부르면 실제로 결과가 나오나요?",
    "header": "자가진단 1/3",
    "options": [
      {"label": "나온다 — Block 4에서 확인했다", "description": "동작하는 스킬 1개 확보. 오늘 산출물 합격"},
      {"label": "부르면 되는데 결과가 얇다", "description": "0.5→1의 0.7쯤. 슬롯을 더 구체화하면 채워진다"},
      {"label": "아직 안 불린다", "description": "SKILL.md description이 모호할 수 있다 — '언제 쓰는 스킬인가' 한 줄 확인"}
    ],
    "multiSelect": false
  }]
})
```

> 피드백: 슬롯 3곳을 채웠고 한 번이라도 결과가 나왔으면 오늘 목표 달성이다.

### 자가진단 2

```json
AskUserQuestion({
  "questions": [{
    "question": "오늘 관통 메시지 — 왜 AI가 잘 시키는 사람에게 더 잘 반응할까요?",
    "header": "자가진단 2/3",
    "options": [
      {"label": "입력이 명확할수록 결과가 명확해서 (GIGO)", "description": "정답. 모호한 입력엔 모호한 결과"},
      {"label": "AI가 원래 알아서 다 해서", "description": "Block 1에서 봤듯, 모호하게 시키면 아무거나 나온다"},
      {"label": "비싼 모델을 써서", "description": "모델보다 '무엇을 어떻게 시키는가'가 결과를 가른다"}

  }]

```

> 피드백: 1번. **일 잘 시키는 사람이 AI도 잘 시킨다** — 오늘 스킬에 되묻기(Clarify)와 내 맥락(슬롯)을 미리 박아둔 이유다.

### 자가진단 3

```json
AskUserQuestion({
  "questions": [{
    "question": "0.5→1 커스터마이징에서 '1'을 만든 건 무엇이었나요?",
    "header": "자가진단 3/3",
    "options": [
      {"label": "슬롯 3곳(세그먼트·톤·금지어)을 내 맥락으로 바꾼 것", "description": "정답. 2곳은 그대로 두고 필요한 곳만 손댔다"},
      {"label": "템플릿을 처음부터 다시 쓴 것", "description": "0.5판을 살리는 게 핵심 — 다시 쓰면 시간만 든다"},
      {"label": "슬롯을 전부 지운 것", "description": "슬롯은 채우는 자리지 지우는 자리가 아니다"}

  }]
})
```

> 피드백: 1번. 남의 템플릿을 내 스킬로 만든 건 **슬롯 3곳**이었다. 이 3곳이 "누구의 스킬인가"를 정한다.
